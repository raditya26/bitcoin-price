<h1> How to run: </h1>
<code>python bitcoin.py [number]</code>
<br>
<br>
<h2>Example:</h2>
<h6>Using int</h6>
Run     : <code>python bitcoin.py 1</code><br>
Output  : <code>$85,925.7969</code><br>

<h6>Using float</h6>
Run     : <code>python bitcoin.py 2.5</code><br>
Output  : <code>$171,936.5522</code><br>
